Place your poster image file here as `poster.png` (or `poster.jpg`).

Recommended filename: poster.png
Recommended dimensions for A3 portrait (pixels): 2480 × 3508 (300 DPI). Lower DPI like 150-200 is faster for web previews.

After copying the image into this folder, open `poster.html` in a browser or click the "Campus Poster" link in the sidebar of `index.html` to preview.

If you prefer an SVG poster, save it as `poster.svg` and update the `src` in `poster.html` to `assets/poster.svg` or call me and I'll inline it for better fidelity.

Audio note:
- Drop your campus anthem audio file here and name it `music.mp3`.
- The compact player under the poster expects `assets/music.mp3`. If you use a different name, update the `<source>` in `index.html`.
